Secret Box

Everyone has something that they want to hide or not let people touch it. So I created a secret box where only you can open it with a secret knock. By this, you can put whatever you want inside the box like; money, love letter and etc. And no one will be able to open it unless they know the secret knock.

How does it works:

To open it, you need to knock on the box with the correct secret knock. If you knock the wrong code, the box will move/ run away from you but if you knock the correct code, it will open up the box.
